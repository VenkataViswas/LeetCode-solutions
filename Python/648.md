# 647. Palindromic Substrings

## Solution 1 (time O(n^2), space O(1))

```python
class Solution(object):
    def countSubstrings(self, s):
        """
        :type s: str
        :rtype: int
        """
        ans = 0
        for i in range(len(s)):
            p1 = p2 = i
            while p1 >= 0 and p2 < len(s) and s[p1] == s[p2]:
                ans += 1
                p1 -= 1
                p2 += 1
        for i in range(len(s) - 1):
            p1, p2 = i, i + 1
            while p1 >= 0 and p2 < len(s) and s[p1] == s[p2]:
                ans += 1
                p1 -= 1
                p2 += 1
        return ans
```
